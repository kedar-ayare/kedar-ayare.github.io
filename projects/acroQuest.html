AcroQuest

Description:

An app that lets you learn about Acronyms.

Problem Statement:

To create an app which assists users to search for Acronyms or Full forms and learn about them.

Features:

Acronym Searching, Acronym Submission, Education

Technology Used + Development Process:

The App is made using React Native while the backend is made using NodeJs (Express Js). The app uses MongoDB for storing all the user data.

A user can easliy register themselves on the app. The user is then guided to a clean Home Screen where he/she can type the acronym to search in the search bar. The App then fetches all matching records from backend and displays it to the user.

The app, not only matches the entered text with the main "acronym" text of the previously stored records, but also with their full forms and descriptions so as to get better results in situations when users enters similar words instead of the exact keyword.
