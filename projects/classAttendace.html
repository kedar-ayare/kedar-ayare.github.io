Class Attendace - Face Recognition

Description:

An app that tracks class attandance using Face Recognition.

Problem Statement:

To create an application that can detect faces from a picture, identify faces, and mark attendace for the same.

Features:

Attendace Marking, Face Recognition

Flow Diagram:

Technology Used + Development Process:

The Face Recognition system makes use of the face_recognition python library to perform maximum amount of tasks. The library helps to convert faces to a set of encodings where each encoding represents a spectific feature of the face like distance between the eyes, or length of forehead or the distance between lips and tip of the nose.

These encodings can be saved in a file system for further reference in the system. In our project, for a single person, we have used his/her multiple images to get the encodings which are then averaged before storing.

The backend of the system is created using Python Flask framework. When the endpoint recieves an image, it saves is to a temporary folder on the server and passes namepath to the face recognition code. The face recognition code then gets encodings of all faces in the image and matches it with the encodings already present in the system. It then sends a list of all matched faces back to the frontend.

The front end is a simple Flutter app that makes use of the system camera to assist user in capturing images which can be then send to the backend for processing.
Once completed, user can see a list of present and absent students.

Challenges Faced:

The main challenge faced was to tunnel the backend running on our local machine since the app was running on our mobild devices.

For this purpose, we used a tool called NGrok, which tunnels a port of the local machine such that it can be accessed from anywhere using internet and hence our app could sent requests to our backend.

Results:

The App provides an easy and clean interface for the user.

The face recognition process detects faces based on object detection and weight calculation, hence results are affected by the picture quality which can be limited by the camera specification of the device.
